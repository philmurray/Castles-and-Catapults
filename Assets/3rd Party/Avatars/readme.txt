This package contains a set of male icons for a fantasy-styled RPG game. It includes 42 icons, 3 borders and 1 icon for random character. You can find 7 different types of characters here: human, elves, orcs, goblins, cat-people, gnomes and mages. All these icons were made with love and passion by Combo21.com team. Current package contains latest version (2.0), which doubled amount of avatar icons by introducing female characters icons.
